Blog questions and answers are given below :

Questions-1 : Discuss the scope of `var`, `let`, and `const`

Ans :
1. var is scope that is hoisted and it value can be reassigned.
2. let is a scope that's value can't be reassigned and it works within the block of curly brackets {} and its value can be reassigned.
3. const is a scope as similar as let but the difference is it's value can't be reassigned.

Questions-2 : Tell us the use cases of `null` and `undefined`

Ans :
1. null is a case when a variable is declared and its value is given as null. It means the default value of the vairable is null.
2. undefined is a case when a variable is declared but it's value is never given. When we try to print that declared variable we get undefined.
